This project is a library written for C that parses the arguments given to the program when executing it from the command-line into a nicely accessible form. It supports flags (also called options) in the style of linux commands (like: ls -la). You can find the GitHub Repository here: https://github.com/HannesLeonha/Vexillum. If you are wondering about the name, it is actually latin and stands for flag.
